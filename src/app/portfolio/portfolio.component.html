Bets Tracker & Analyser

   Project idea creator and developer.
I worked on this project for 3 years developing Bets Tracker and Bets Analyser in WinForms using C#, Entity Framework and SQL Db. One part of data used in application is parsed from web, and other part is calculated from that data and saved in database tables.
   Main Idea of Bets Tracker is to track your bets everyday with easy data entry UI, edit and delete bets and track your overall wins and loses.
   In Bets Analyser you can see data like last 50 games, games between current game opponents, points per game, ATS, SU and Over/under stats.
Also, you can see current live games lines and results if games are in progress.
By clicking on analyse current game, you get results from similar games and application predicts for you which bet to take (e.g. 85% on Over).

BetsKiller

   Project idea creator and developer.
This project is a continuation on Bets Tracker and Analyser project which was made in WinForms, BetsKiller is made in ASP.NET MVC using Entity Framework and JSON APIs to recieve and parse data from the reliable stats source.
   BetsKiller consists of many views, among which we have Dashboard with various stats and info about games and offers. BetsTracker page to track your bets, Sports picks, Team Stats, Leaders, Standings etc.
